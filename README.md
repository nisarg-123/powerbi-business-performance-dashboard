# powerbi-business-performance-dashboard
Interactive Power BI dashboard for business performance analysis using DAX and data modeling
## Project Overview
This project analyzes business performance using Power BI with interactive dashboards.

## Key KPIs
- Total Revenue
- Month-over-Month Growth
- Region-wise Performance
- Category Contribution

## Tools & Technologies
- Power BI
- DAX
- Excel

## Key Insights
- Identified top-performing regions
- Seasonal trends in revenue
- Underperforming product categories
